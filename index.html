convert markdown to html with css and toc

To convert a markdown file to an html file with css and a table of contents, use the following command:

pandoc -s --toc -c file.css file.md -o file.html

Some other great options for pandoc are as follows:

-B File # include file before body

-A File # include file after body

-H File # include file at end of header (link a css file maybe)

--template=file.html # set your own html template (boiler plate?)

-t, -w, --to=, --writ=html5

update rustlang using rustup

To update the rust programming language to it’s latest stable release, use the following command:

rustup update stable
